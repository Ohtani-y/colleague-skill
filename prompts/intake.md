# 基本情報入力スクリプト

## 冒頭の挨拶

```
この同僚のSkillを作成します。3つの質問に答えるだけでOKです。どれもスキップ可能です。
```

---

## 質問シーケンス

### Q1：ニックネーム／コードネーム

```
この同僚はどう呼ばれていますか？（ニックネーム、あだ名、コードネームなど。複数語は - で繋げてください）

例：taro-yamada
```

- 任意の文字列を受け付ける
- 生成するslugは統一して `-` で繋ぐ（アンダースコアは使わない）
- 日本語は自動的にローマ字に変換して `-` で繋ぐ（「山田太郎」→ `taro-yamada`、「小林」→ `kobayashi`）
- 英語はそのまま小文字で `-` 繋ぎ（"Big Mike" → `big-mike`）

### Q2：基本情報

会社、職級、職位、性別を1つの質問にまとめて、一言で回答してもらう：

```
一言で基本情報を教えてください——会社、職級、職位、性別など、思いつくものを何でも。スキップもOKです。

例：ByteDance 2-1 バックエンドエンジニア 男性
```

ユーザーの回答から以下のフィールドを解析する（不明な項目は空欄）：
- **会社**
- **職級**
- **職位**
- **性別**

### Q3：性格プロフィール

MBTI、星座、個性タグ、企業文化タグ、主観的印象をすべてまとめて、自由に記述してもらう：

```
一言で性格を教えてください——MBTI、星座、個性の特徴、企業文化の影響、あなたから見た印象など、
思いつくものを何でも。スキップもOKです。

例：INTJ やぎ座 責任転嫁の達人 ByteDance流 CRは厳しいけど理由を説明しない
```

ユーザーの回答から以下のフィールドを識別・抽出する（不明な項目は空欄）：
- **MBTI**：16種の標準タイプ
- **星座**：12星座
- **個性タグ**：下記タグライブラリからマッチング、カスタム記述も受け付ける
- **企業文化タグ**：下記タグライブラリからマッチング
- **主観的印象**：分類できない自由記述はそのまま原文を保持

#### 個性タグライブラリ

**仕事への姿勢**：真面目で責任感が強い / まあまあでOK / 責任転嫁の達人 / いつも尻拭い役 / 完璧主義 / 先延ばし癖

**コミュニケーションスタイル**：ストレート / 遠回し / 寡黙 / おしゃべり / 音声メッセージ好き / 既読スルー / 既読で的外れ返信 / 即レス強迫症

**意思決定スタイル**：即断即決 / 二転三転 / 上司頼り / 強引に推進 / データドリブン / 完全に直感

**感情スタイル**：メンタル安定 / 打たれ弱い / すぐ興奮 / 冷淡・無関心 / 表面上は穏やか / 皮肉・嫌味

**話術・手段**：パワハラ気質 / 社内政治プレイヤー / 責任転嫁アーティスト / 上司マネジメントの達人 / 説教好き / 感情で操る

#### 企業文化タグライブラリ

- **ByteDance流** — 率直でストレート、インパクト重視、何でもまずcontextを話す、「認識を合わせる」が口癖
- **Alibaba流** — 社内用語「エンパワーメント」「エコシステム」「クローズドループ」を多用
- **Tencent流** — データ重視、A/Bテスト文化、保守的、ユーザー体験最優先
- **Huawei流** — 奮闘者文化、プロセス重視、PPT報告が得意、実行力を強調
- **Baidu流** — 技術至上主義、階層意識が強い、社内競争が激しい
- **Meituan流** — 徹底した実行力、細部にこだわる、現場主義
- **第一原理主義** — イーロン・マスク式、本質を追求、類推を拒否、過激な簡素化
- **OKR信奉者** — 何でもまずObjectiveを確認、KRの粒度にこだわる
- **大企業マニュアル人間** — SOP依存、想定外に弱い、安定性は高いが創造性は低い
- **スタートアップ気質** — リソース制約下で取捨選択、何でも自分でやる、混沌に強い、結果重視
